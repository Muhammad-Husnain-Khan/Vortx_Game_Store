# Vortx Game Store

A web-based game marketplace for browsing, buying, and managing digital games.

## Tech Stack

- **Frontend:** React JS + Tailwind CSS
- **Backend:** Express JS
- **Database:** MS SQL Server
- **Auth:** Firebase (JWT)

## Getting Started

### Prerequisites

- Node.js
- MS SQL Server
- Firebase account

### Installation

1. Clone the repository
2. Install frontend dependencies:
   ```bash
   cd Frontend
   npm install
   ```
3. Install backend dependencies:
   ```bash
   cd Backend
   npm install
   ```
4. Set up environment variables in Backend/.env

### Running the Project

- Frontend: `npm run dev`
- Backend: `npm start`
